## 明确不能臆造的事实

以下信息必须查证，不能由 Agent 编造：

- [ ] Star 数（必须查 GitHub 页面）
- [ ] 下载量（必须查 npm/pypi 等平台）
- [ ] 用户数量（必须有人工提供）
- [ ] 性能数据（必须有测试依据）
- [ ] 外部依赖版本号（必须查证）
- [ ] 发布时间（必须有人工确认）
- [ ] 他人贡献（必须注明来源）

## 使用说明

1. 复制此模板到 `drafts/sources/<项目名>/source.md`
2. 尽可能填写完整，不确定的字段标注 [待确认]
3. 标注「不能臆造」的信息需要人工验证后填写
4. 整理完成后通知 Agent 开始写文章
